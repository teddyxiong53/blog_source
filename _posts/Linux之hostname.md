1

hostname哪些地方有：

```
/etc # cat hostname
rockchip
/etc #
/etc # cat /proc/sys/kernel/hostname
rockchip
/etc # sysctl kernel.hostname
kernel.hostname = rockchip
```

如果修改了hostname，会导致我正在运行的程序调用了gethostbyname失败。

```
gethostbyname fail, reason:Connection timed out
```
