# 书籍

《Learning Python》（中文名《python学习手册》）

第四版。重点是Python3.0 。说明跟python2.6的不同。

重点关注语言本身。

《Programming Python》

跟《Learning Python》是一个系列的。

重点关注应用。

作者是做python培训的。这些内容都是总结他的培训经验。

《Learning Python》是3天的培训课程。



《疯狂python讲义》。李刚的疯狂讲义系列的。都比较实用。
