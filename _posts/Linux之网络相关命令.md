对linux下网络相关命令，建立一个全局性的认识。

```
网络配置相关：
	ping
	ip
路由相关：
	netstat
	route
	ip
查看端口工具：
	netstat
	lsof
	ss
	nc
	telnet
下载工具：
	curl 
	wget
	axel
防火墙：
	iptables
流量相关：
	iftop
	nethogs
连通性及响应速度：
	ping
	traceroute
	mtr
	tracepath
域名相关：
	nslookup
	dig
	whois
抓包：
	tcpdump
网桥相关：
	ip
	brctl
	ifconfig
	ovs
```

除了本来就很熟练的，重点掌握nc和ip这2个。
