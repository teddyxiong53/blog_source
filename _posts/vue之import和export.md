1

Vue 是通过 webpack 实现的模块化，因此可以使用 import 来引入模块

**引入的时候可以给这个模块取任意名字**，例如 "obj"，且不需要用大括号括起来。

```
import mInput from "m-input.vue"
```

mInput可以不用mInput，随便写都行。只是我们一般遵循规范。
