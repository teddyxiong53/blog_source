# 插入图片

```
1、头部写上这个，保证可以实时在typora上预览到图片。
typora-root-url: ..\
2、图片举例。
![xx](/images/xx.png "xx")
```



特殊格式的设置，基本是靠html语法来做的。

尤其是公司文档的封面信息，要靠html来实现灵活的布局。

```
表格里换行
```
