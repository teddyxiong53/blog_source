总是需要频繁提交内容到github上，输入用户名密码比较繁琐。按照网上的方法试了几种都没有效果。

现在要说的这种是亲测有效的。很简单。

tortoisegit 
setting --> Git --> Credential -->在下拉从菜单里选择"manager -- all windows users"。就好了。
