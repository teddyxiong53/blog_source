1

http头部的作用是什么？

允许客户端和服务端在头部里添加附加信息。

组成是键值对。

http的头部有哪些内容？



根据上下文的不同，消息头部分分类：

```
1、一般头。
2、请求头。
3、响应头。
4、实体头。
```

根据代理对头部的处理方式，又可以分为：

```
1、端到端消息头。
	代理也必须要转发这部分。
2、逐跳消息头。
	代理不转发。
```
