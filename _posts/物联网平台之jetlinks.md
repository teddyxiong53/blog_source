下载代码：

```
git clone https://gitee.com/jetlinks/jetlinks-community
```

使用docker方式来启动。

```
$ cd docker/run-all
$ docker-compose up
```

特别占用资源，我的服务器一运行，CPU和ram就100%了。

先把教程过一遍，

访问地址：

地址: http://localhost:9000, 用户名:admin,密码:admin.
