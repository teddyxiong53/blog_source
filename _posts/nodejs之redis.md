1

使用ioredis这个模块。

set和get。

```
var Redis = require("ioredis")
var redis = new Redis()
redis.set("foo", "bar")
redis.get("foo", function(err, result) {
    console.log(result)
})
redis.del("foo")
```

如果get函数里，最后一个参数不是回调，那么返回一个promise。

```
redis.get("foo").then(function(result) {
    console.log(result)
})
```
