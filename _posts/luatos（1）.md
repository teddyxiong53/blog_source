# 入口代码流程研究

入口文件是：

bsp\linux\src\main.c

分配了1M的内存：

```
#define LUAT_HEAP_SIZE (1024*1024)
uint8_t luavm_heap[LUAT_HEAP_SIZE] = {0};
```

```
bpool(luavm_heap, LUAT_HEAP_SIZE);
```

这个属于lua5.3.6较新版本新增的内容了。
