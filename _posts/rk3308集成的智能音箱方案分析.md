因为当前系统是基于64位进行编译，所以先把64位的都跑一遍。

aispeech-2mic-64bit

```
修改配置：
1、MIC_NUM改为2.
2、oem目录改为aispeech-2mic-64bit
就改这2点。
```

配置是在BoardConfig.mk里。

所以，编译都这样执行：

```
1、拷贝对应的配置文件为BoardConfig.mk。
2、执行build.sh脚本。
```

编译后全部烧录。

速度足够快。保险起见，都烧录。

烧录后，自动运行，提示“我没有没有联网呢，请通过手机app为我联网”。
