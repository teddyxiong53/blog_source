# pocketsphinx离线stt引擎

要使用pocketsphinx（下面简写为ps）。需要安装这些组件。

1、sphinxbase和pocketsphinx。

这个直接用apt-get来安装就可以了。

```
sudo apt-get install pocketsphinx 
```

2、cmuclmtk。

这个是语言模型训练工具。

3、m2m-aligner

4、openfst。

5、is2013-conversion。

6、mitlm。
