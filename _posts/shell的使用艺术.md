# 日常使用

shell输入时的删除：

```
ctrl+w：删除光标前面的一个单词。
ctrl+u：从光标处到最前面都删掉。
ctrl+k：从光标到行尾的都删掉。
ctrl+l：相当于clear。但是这个方便多了。
```

man readline，可以看到很多快捷键。



如果你输入了一条很长的命令，但是现在不想执行，有个窍门可以保存。

就是在前面加上#号执行一下，后面就可以在history里找到了。

pstree -p。可以用树形结构来展示进程的关系。



#实用小工具

printenv：打印环境变量。

cut/paste/join：数据修改。

expand和unexpand：空格和tab转换。

seq：打印一个范围的所有自然数。

slurm：查看网速，很形象。

watch：后面跟一个命令，会高亮显式不同的部分。自动反复执行后面的命令。
