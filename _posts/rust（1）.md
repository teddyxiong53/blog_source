# rust环境搭建

https://www.rust-lang.org/zh-CN/tools

用rustup来安装rust。

https://github.com/rust-lang/rustup

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- --default-toolchain none -y
```

然后

```
source "$HOME/.cargo/env"
```
