# 基本用法

1、本地化的工具，是一个使用本地 127.0.0.1:8888 的 HTTP 代理
（任何能够设置 HTTP 代理为 127.0.0.1:8888 的浏览器和应用程序都可以使用 Fiddler）

2、Fiddler本身对http协议支持较好，且操作简单容易上手

3、具有抓包和分析功能，省去了安装其他工具的必要

4、工具本身功能强大，如劫包、篡改数据、限速等
