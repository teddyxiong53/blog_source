还是要把前段的框架学习一下，就以vue作为学习对象。

首先，需要把开发环境搭建好。

都在linux下做。这样环境会比较干净。

全新安装一个kubuntu16.40的。

```
npm install -g vue-cli 
```

新建项目。

```
vue init webpack vue-demo
```

运行：

```
npm run dev
```



```
http://localhost:8080
```
