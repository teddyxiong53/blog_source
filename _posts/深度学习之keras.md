什么是keras？

keras是一个神经网络库，封装了tensorflow、CNTK、Theano这些库，提供简单易用的api。

主要是为了快速实验用。

可以帮助你快速实现自己的想法。

特点：

```
1、允许简单而快速的原型设计。
2、同时支持卷积神经网络和循环神经网络。
3、在cpu和gpu上无缝运行。
```

设计原则是：

```
1、对用户友好。api尽量简单。
2、模块化。
3、容易扩展。
4、基于python实现。
```



keras的核心数据结构是model。

model表示一种组织网络层的方式。
