需要找一个electron开发的实际例子来看看electron的写法。

看了一下官方的列表里，发现这个新闻客户端比较适合，因为逻辑简单。

https://electronjs.org/apps/headlines

下载安装运行：

```
git clone https://github.com/MedZed/Electron-Headlines.git
cd Electron-Headlines
npm install
npm start
```
