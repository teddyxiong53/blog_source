tekUI 是一个轻量, 独立, 可移植的 GUI 工具包, 

用 lua 和 C 开发. 

最初是为 X 窗口系统开发, 

后来移植到 Linux framebuffer, raw framebuffer, DirectFB, Windows 和其他窗口系统.



tekUI 不是已有 GUI 库的绑定,

 不打算与原生 GUI 的外观风格保持统一(但可以通过模拟达到一定程度的相似). 

尽管是用脚本语言编写的, 但实现相当高效, 资源占用低. 

**对开发者的好处是简洁, 透明和模块化.** 

可以很轻松地增加定制界面元素.
