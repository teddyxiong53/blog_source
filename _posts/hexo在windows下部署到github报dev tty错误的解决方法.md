在windows下部署hexo到github，总是报/dev/tty找不到设备。
我的博客的_config.yml文件的部署部分是这样的：
```
deploy:
	type: git
	repo: https://github.com/teddyxiong53/teddyxiong53.github.io
	branch: master
```

折腾了好久，终于发现只要这样改就好了。
```
deploy:
	type: git
	repo: git@github.com:teddyxiong53/teddyxiong53.github.io.git
	branch: master
```

然后再hexo d，可以看到部署成功了。
